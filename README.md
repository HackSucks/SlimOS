# SlimOS
# What is it?
SlimOS is a lightweight Linux OS based off of Lubuntu
# What does it have?
Firefox, Libreoffice, and LXDE as the DE and Openbox as the WM
# How do I install?
grab the .iso, let it boot, and run setup.sh, make sure it is executable.
